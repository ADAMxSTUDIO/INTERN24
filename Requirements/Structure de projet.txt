Structure recommandée pour votre projet:

Pour une application complete de gestion des demandes de congés avec notifications, voici ce qui
est recommandé :
1. Table users : Gestion des utilisateurs avec röles.
2. Table leave_requests : Gestion des demandes de congés.
3. Table notifications : Gestion des notifications envoyées aux utilisateurs (par email et dans la
base de données).
4. Table password_resets : Réinitialisation de mot de passe.
5. Table : Pour l'authentification API via Laravel Sanctum.
6. Table failed_jobs (optionnelle) : Si vous utilisez des jobs pour les notifications par email ou
d'autres fonctionnalités.
